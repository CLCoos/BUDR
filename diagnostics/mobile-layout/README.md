# Mobil layout-diagnose (iPhone 13)

Genereret af `e2e/mobile-overflow-diagnose.spec.ts`.

**Kør igen:**

```bash
PLAYWRIGHT_BROWSERS_PATH=.playwright-browsers npx playwright test e2e/mobile-overflow-diagnose.spec.ts
```

**Enhed:** 390×844, `deviceScaleFactor: 3`, touch (Chromium + iPhone 13 user-agent).

**Filer:**

- `report.json` — fuld maskin-læsbar rapport
- `iphone13-*-viewport.png` — det brugeren ser på skærmen
- `iphone13-*-full.png` — fuld side (scroll)
